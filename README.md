# Akshita Sure Portfolio

A modern portfolio website built with **Next.js**, **Tailwind CSS**, and **TypeScript**.

View the live site: https://portfolio-akshita.vercel.app

## Getting Started

### Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/akshitasure12/portfolio-akshita.git
cd portfolio-akshita
npm install
```

### Running Locally

Start the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
